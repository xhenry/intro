Responsive Design Components

The responsive design features of the web site changes the web site appearance based on the size of the viewport. Desktops have four panels displayed. Laptops have two panels displayed. Mobile phones display one panel and a collapsed menu. The Contact Us section of the web site keeps the form components in a similar orientation for each device type, mobile, laptop and desktop.

Responsive Images

The responsive image features of the web site changes the web site based on the size of the viewport of the device being used. This means the web site will send small images to small devices and larger images to devices with larger viewports. Image size may be verified because each image has a label on the lower left hand corner which describes the size of the image. Just below the carousel the user will get viewport size information each time the viewport passes a media break point.

Bootstrap Components

Bootstrap features used in demo web site include: collapsable menu, carousel with four images, four panels, two modals, two tool tips, four accordians, a form with five components. The arragment of these components change with changes in viewport size.

Technology

The web site uses HTML to structure the web site and CSS to change the styling of it’s components. Web site functionality depends on the use of JavaScript and jQuery. Most notable JavaScript features include: the responsive loading of carousel images, menu class changes and the tool tips. Gulp was used to minify the CSS and JavaScript used on this demo. Git was used to load the web site onto GitHub for web site hosting.
